# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across different product categories. The objective is to identify spending patterns, customer segments, product preferences, and subscription behavior.

## Tools Used
Python (Pandas)
PostgreSQL
Power BI

## Dataset
Rows: 3900
Columns: 18

Features include:
- Customer demographics
- Purchase details
- Shopping behavior

## Key Analysis
- Revenue by Gender
- Top 5 Products by Rating
- Shipping Type Comparison
- Customer Segmentation
- Top 3 Products per Category
- Subscription vs Non-Subscription Spending

## Dashboard
Power BI dashboard visualizes customer insights including revenue trends, category performance, and customer segmentation.

## Business Recommendations
- Increase subscription benefits
- Launch loyalty programs
- Improve product promotion for top-rated products
