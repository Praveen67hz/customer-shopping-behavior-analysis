📊 Customer Shopping Behavior – Data Analytics Project

📌 Overview

This project analyzes customer shopping behavior using a complete data analytics workflow. The dataset is processed using Python (Pandas) for data exploration and cleaning, analyzed using SQL, and visualized through a Power BI dashboard.

The goal of the project is to extract meaningful insights about customer purchasing patterns, product performance, and sales trends.

This project demonstrates the end-to-end data analysis process, including:

• Data loading and exploration
• Data cleaning and preprocessing
• SQL-based data analysis
• Data visualization
• Insight reporting


📂 Dataset

The dataset contains information related to customer purchases in an online shopping environment. It includes fields such as:

• Customer ID
• Age
• Gender
• Item Purchased
• Category
• Purchase Amount
• Location
• Payment Method
• Shipping Type
• Frequency of Purchases

This dataset is used to analyze customer behavior and identify patterns in purchasing decisions.


🛠 Tools & Technologies

The following tools were used in this project:

• 🐍 Python (Pandas) – Data loading, cleaning, and exploratory data analysis (EDA)
• 📓 Jupyter Notebook – Running Python analysis
• 🗄 SQL (PostgreSQL / MySQL / SQL Server) – Data querying and analysis
• 📊 Power BI – Interactive dashboard and visualization
• 🎤 Gamma – Presentation slides for project summary


⚙️ Project Steps

📥 1. Data Loading
The dataset was loaded into Python using Pandas for initial analysis.

🔎 2. Exploratory Data Analysis (EDA)

Basic exploration was performed to understand:

• Data structure
• Missing values
• Data distributions
• Key statistics


🧹 3. Data Cleaning

Data cleaning included:

• Handling missing values
• Fixing data types
• Removing inconsistencies
• Preparing the dataset for analysis


🗃 4. SQL Analysis

The cleaned data was analyzed using SQL queries to answer business questions such as:

• Most purchased product categories
• Customer purchase patterns
• Sales distribution by location
• Top spending customers


📊 5. Data Visualization

A Power BI dashboard was built to visually present insights from the dataset.


📈 Power BI Dashboard

The dashboard includes visualizations such as:

• Total Purchase Amount
• Sales by Category
• Purchase Distribution by Gender
• Payment Method Analysis
• Customer Demographics
• Purchase Trends

The dashboard helps stakeholders quickly understand customer behavior and key sales patterns.


🔍 Key Insights / Results

Some insights identified from the analysis include:

• Certain product categories have significantly higher purchase volumes.
• Customer purchasing behavior varies across age groups and gender.
• Payment method preferences differ among customers.
• Some locations show higher purchasing activity than others.

These insights can help businesses improve marketing strategies and customer targeting.


🎯 Presentation

A project presentation was created using Gamma to summarize:

• Project objective
• Analysis process
• Key findings
• Business insights

This helps communicate results clearly to stakeholders.


▶️ How to Run the Project

1. Download the dataset.
2. Open the Jupyter Notebook.
3. Install required libraries:

pip install pandas numpy

4. Run the notebook to perform:
   • Data loading
   • Data cleaning
   • Exploratory data analysis

5. Execute SQL queries in your SQL environment (PostgreSQL / MySQL / SQL Server).
6. Open the Power BI file to view the dashboard.
